API documentation

About

We provide an API for you to use your visualization on your website or Microsoft PowerPoint presentation.


How to use our API ?


On the result page, after you have generated a visualization. You will see a link below your visualization.

This is your visualization API Url. You can click "copy" to copy the link to your clipboard. After that, you can use it in :

  • Website
  • Microsoft PowerPoint


How to use the API on your website ?

How to use the API on your Microsoft PowerPoint presentation ?


You will need to get "Web Viewer" add-in.
On toolbar, click "insert" then "Get Add-ins" after that search for "Web Viewer".

Once you added Web Viewer, you can add Web Viewer add-in on your presentation slide then place the API Url to show the visualization.
